# 🚀 Google AI Studio로 MVP 앱 만들기 — 완전 가이드

> **대상:** 기업 실무자 (비개발자) | **소요 시간:** 1~2시간 | **난이도:** ★☆☆☆☆
> **필요한 것:** 브라우저 하나! 설치할 프로그램 없음.

이 가이드는 **코딩 경험이 없는 기업 실무자**가 AI의 도움을 받아 업무용 웹 앱을 만들고 팀에 배포하는 전 과정을 담습니다. 복사 붙여넣기와 클릭만으로 완성할 수 있습니다.

---

## 📦 이 가이드로 만드는 것

**직장인 점심 & 음료 신청 앱**

| 항목 | 설명 |
|------|------|
| AI 앱 빌더 | Google AI Studio Build 탭 (코드 자동 생성) |
| 데이터베이스 | Firebase Firestore (실시간 클라우드 DB) |
| 로그인 | Google 계정 로그인 (직원용 인증) |
| 배포 URL | Vercel로 팀 전체 공유 가능한 웹 주소 |

---

## 🗺️ 전체 흐름

```
[Step 0] 계정 3개 준비 (Google + GitHub + Vercel) — 5분
     ↓
[Step 1] AI Studio Build에서 앱 설명 입력 → 코드 생성 → GitHub 커밋 — 15분
     ↓
[Step 2] Firebase 프로젝트 생성 + 설정값 복사 — 15분
     ↓
[Step 3] Vercel에서 GitHub 저장소 연결 → 배포 — 10분
     ↓
[완성! 🎉]
```

## ❓ 자주 묻는 질문

**Q: 비용이 드나요?**
A: Google AI Studio, Firebase(Spark 플랜), Vercel(Hobby 플랜) 모두 무료로 시작할 수 있습니다. 사용량에 따라 유료 전환이 필요할 수 있습니다.

**Q: Node.js나 VS Code를 설치해야 하나요?**
A: 아니요! 이 가이드는 브라우저만으로 완성할 수 있습니다. Vercel이 서버에서 빌드를 대신 해줍니다.

**Q: API 키를 발급해야 하나요?**
A: 아니요! 이 가이드의 점심 신청 앱은 Firebase만 사용합니다. Gemini API 키가 필요 없어요.

**Q: 혼자 하기 어려우면?**
A: AI Studio 채팅창에 막히는 상황을 그대로 입력하면 바로 도움을 받을 수 있습니다. 예: "로그인이 안 돼", "데이터가 저장이 안 돼"
