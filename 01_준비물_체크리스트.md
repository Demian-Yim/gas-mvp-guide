# ✅ Step 0: 시작 전 준비 — 계정 3개만 만들면 됩니다

> **브라우저만 있으면 됩니다!** 설치할 프로그램은 없어요.

---

## 필요한 계정 3개

모두 **완전 무료**입니다. Google 계정 하나로 나머지도 만들 수 있어요.

| 계정 | 무엇에 쓰나요? | 만드는 방법 |
|------|--------------|------------|
| **Google 계정** | AI Studio + Firebase | 이미 있으면 그냥 넘어가세요 ✅ |
| **GitHub 계정** | 앱 코드 보관 창고 | [github.com](https://github.com) → Sign up 클릭 |
| **Vercel 계정** | 인터넷에 앱 올리기 | [vercel.com](https://vercel.com) → Continue with GitHub 클릭 |

> 💡 **팁:** Vercel은 GitHub 계정으로 가입하면 자동 연결됩니다. GitHub 먼저 만드세요!

---

## 가입 순서 (이 순서대로 하세요)

### 1단계: GitHub 가입

1. [github.com](https://github.com) 접속
2. 우측 상단 **"Sign up"** 클릭
3. 이메일 입력 → 비밀번호 설정 → 사용자 이름 입력
4. 이메일로 온 인증 코드 입력

✅ **성공 확인:** GitHub 대시보드 화면이 나타납니다.

---

### 2단계: Vercel 가입

1. [vercel.com](https://vercel.com) 접속
2. **"Sign Up"** 클릭
3. **"Continue with GitHub"** 클릭 (GitHub 계정으로 자동 가입)
4. 권한 허용 창 → **"Authorize"** 클릭

✅ **성공 확인:** Vercel 대시보드가 열립니다.

---

## ✅ 준비 완료 체크리스트

아래 3개 모두 체크하면 다음 단계로 이동하세요!

- [ ] Google 계정 (Gmail) — 로그인 상태 확인
- [ ] GitHub 계정 — 대시보드 열린 상태
- [ ] Vercel 계정 — GitHub으로 가입 완료
