# 🏗️ Step 1: AI Studio Build — 앱 만들기

> **이 단계에서 할 일:** Google AI Studio의 Build 탭에서 앱 설명을 입력하면 코드를 자동으로 만들어줍니다. API 키나 코딩 지식 없이 클릭과 복붙으로 완성!

---

## 1. AI Studio 접속 & Build 탭 열기

1. 브라우저에서 **[aistudio.google.com](https://aistudio.google.com)** 접속
2. Google 계정으로 **로그인**
3. 좌측 메뉴에서 **"Build"** 클릭
4. **"Create app"** 버튼 클릭

![AI Studio 화면](screenshots/aistudio_landing.png)
*↑ 로그인 후 좌측 메뉴에서 Build를 찾으세요*

---

## 2. 모델 선택

채팅창 위쪽에서 모델을 확인하세요.

- **Gemini 2.0 Flash** 선택 ← 빠르고 충분합니다

---

## 3. 앱 설명 입력하기

채팅창에 아래 내용을 **그대로 복사해서 붙여넣기** 하세요.

```
직장인용 점심 & 음료 신청 앱을 만들어줘.

점심 메뉴: 불고기덮밥, 김치찌개, 파스타, 샐러드
음료: 아메리카노, 라떼, 녹차, 물

화면 2개:
1. 신청 화면: 이름 + 메뉴/음료 선택 + "신청하기" 버튼 → Firebase 저장
2. 관리자 화면: 오늘 전체 신청 목록 + 메뉴별/음료별 집계 숫자

디자인: 깔끔한 흰색 배경, 한국어 UI, 모바일 최적화

Firebase 설정은 나중에 알려줄게. 지금은 아래처럼 빈 값으로 코드 작성해줘:
const firebaseConfig = {
  apiKey: "FILL_IN",
  authDomain: "FILL_IN",
  projectId: "FILL_IN",
  storageBucket: "FILL_IN",
  messagingSenderId: "FILL_IN",
  appId: "FILL_IN"
};
```

> ⏳ **1~2분 기다리면** 우측 패널에 앱 미리보기가 나타납니다.

✅ **성공 확인:** 오른쪽에 신청 폼 + 관리자 화면이 보이면 완성!

---

## 4. 마음에 안 드는 부분 수정하기

채팅창에 말로 요청하면 바로 수정됩니다.

| 원하는 변경 | 이렇게 입력하세요 |
|------------|----------------|
| 메뉴 추가 | `점심 메뉴에 "비빔밥"도 추가해줘` |
| 집계 방식 변경 | `관리자 화면에 그래프도 추가해줘` |
| 디자인 수정 | `버튼을 더 크고 둥글게 바꿔줘` |
| 기능 추가 | `한 사람이 하루 1회만 신청 가능하게 해줘` |

> 💬 막히면 그냥 현상을 그대로 채팅창에 입력하면 AI가 직접 고쳐줍니다.
