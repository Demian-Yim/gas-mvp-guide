# 📤 Step 1 (계속): GitHub에 코드 올리기

> **이 단계에서 할 일:** AI Studio에서 만든 앱 코드를 "Sync to GitHub" 버튼 한 번으로 GitHub에 올립니다. 다운로드나 터미널 없이 클릭만으로 완성!

---

## 1. Sync to GitHub 버튼 클릭

앱 미리보기가 완성되면:

1. 화면 **오른쪽 위**에서 **"Sync to GitHub"** 버튼 클릭

![Sync to GitHub 버튼](screenshots/step1_gas_github_button.png)
*↑ 오른쪽 상단 — "Sync to GitHub" 버튼*

---

## 2. 처음 사용 시: GitHub 계정 연결

2. GitHub 계정 연동 팝업이 뜨면 → **"Authorize"** 클릭

![GitHub 연결](screenshots/step1_gas_github_auth.png)
*↑ "Authorize Google AI Studio" 클릭 → AI Studio와 GitHub 연결 (처음 1회만)*

---

## 3. 저장소 이름 입력

3. 저장소 이름 입력 (예: `lunch-cafe`) → **"Create and push"** 클릭

![저장소 이름 입력](screenshots/step1_gas_repo_name.png)
*↑ 이름 입력 → "Create and push" → 내 GitHub에 저장소 생성됨*

> 💡 **이름 규칙:** 영어 소문자 + 하이픈만 사용 (예: lunch-cafe, office-menu, team-meal-app)

---

## 4. GitHub에서 확인하기

1. [github.com](https://github.com) 접속 → 로그인
2. 우측 상단 프로필 → **"Your repositories"** 클릭
3. 방금 만든 저장소 클릭
4. `App.tsx`, `package.json`, `src/` 폴더가 보이면 완성!

✅ **성공 확인:** 저장소 이름 옆에 **"on main"** 이 표시되면 정상입니다.

---

## 앱 수정할 때는?

1. AI Studio 채팅창에 원하는 수정 입력
2. **"Sync to GitHub"** 버튼 다시 클릭 → 자동으로 최신 코드가 업데이트됩니다
