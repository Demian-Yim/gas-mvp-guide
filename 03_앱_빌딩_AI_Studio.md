# 📤 Step 1 (계속): GitHub에 코드 올리기

> **이 단계에서 할 일:** AI Studio에서 만든 앱 코드를 GitHub에 바로 올립니다. 다운로드나 터미널 없이 버튼 하나로 완성!

---

## GitHub 커밋 버튼 사용하기

AI Studio Build에서 앱 미리보기가 완성되면:

1. 화면 **우측 상단**의 **GitHub 아이콘(🐙)** 버튼 클릭

   > 📍 Export, Download 버튼 옆에 있습니다. GitHub 로고처럼 생긴 버튼이에요.

2. GitHub 계정 연동 팝업이 뜨면 → **"Authorize"** 클릭
3. **저장소 선택** 창에서:
   - 기존 저장소가 있으면 선택
   - 새 저장소를 만들려면 이름 입력 (예: `lunch-app`) → **"Create & Commit"** 클릭
4. 커밋 메시지 확인 → **"Commit"** 클릭

✅ **성공 확인:** GitHub 저장소 링크가 나타납니다. 클릭해서 파일이 올라갔는지 확인하세요.

---

## GitHub에서 확인하기

1. [github.com](https://github.com) 접속 → 로그인
2. 우측 상단 프로필 → **"Your repositories"** 클릭
3. 방금 만든 저장소 클릭
4. `src/`, `package.json` 등 파일이 보이면 완성!

> 💬 **GitHub 연동이 안 된다면?** GAS 채팅창에 "GitHub에 올리는 방법 알려줘"라고 입력하면 안내해줍니다.
